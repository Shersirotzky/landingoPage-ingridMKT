# Landing Page — Ingrid Rafaela

Projeto organizado em pastas, com HTML, CSS e JavaScript separados.

## Estrutura

- `index.html` → estrutura da página
- `assets/css/styles.css` → estilos e responsividade
- `assets/js/main.js` → animações e comportamento dos botões
- `assets/js/config.js` → configuração do link do Google Forms

## Como ativar o botão do formulário

Abra o arquivo:

`assets/js/config.js`

E troque:

`COLE_AQUI_O_LINK_DO_SEU_GOOGLE_FORMS`

pelo link real do seu Google Forms.

Exemplo:

```js
window.APP_CONFIG = {
  googleFormUrl: 'https://docs.google.com/forms/d/e/SEU-LINK/viewform'
};
```
